# Snake Game

## Description

This project is a simple Snake Game built using HTML, CSS, and JavaScript. The game runs in the browser and focuses on core snake mechanics such as movement, growth, and self-collision.

## Demo

Live Link:
https://pranav2020-pixel.github.io/snake-game-2/

## Features

* Smooth snake movement
* Snake grows after eating food
* Random food generation
* Self-collision detection
* Score tracking
* Lightweight and fast gameplay

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

snake-game-2/

## How to Run

1. Clone the repository:

```
git clone https://github.com/pranav2020-pixel/snake-game-2.git
```

2. Open the project folder
3. Run `index.html` in a web browser

## Controls

* Arrow Keys to control the snake

  * Up → Move Up
  * Down → Move Down
  * Left → Move Left
  * Right → Move Right

## Game Rules

* Eat food to increase score
* Snake grows in length after eating
* Game ends when the snake collides with itself
